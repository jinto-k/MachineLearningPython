Libraries
1. Numpy
   * provides multidimensional array
2. Pandas
    * data analysis library that provides data frame.
    * data frame is 2d data structure similar to excel spreadsheet.
3. MathPlotLib
    * 2d plotting library for creating graphs on plots.
4. Scikit-Learn
    * provides common algorithms like decision trees, neural networks,etc.


We use Jupyter Notebook for ML works.

* install anaconda navigator from anaconda.com
* run 'jupyter notebook' on the terminal to launch notebook
* This will open a default dashboard on the browser.
* Then we create a notebook at the desired location
